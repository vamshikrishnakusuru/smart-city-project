# Smart City Management System 🚀

A full-stack web application to report and manage city issues.

## 🔥 Features
- User Registration & Login (JWT Authentication)
- Report Issues (Water, Roads, etc.)
- Track Issues
- Admin Dashboard
- Secure APIs with Spring Security

## 🛠 Tech Stack
### Frontend:
- React (Vite)
- Tailwind CSS

### Backend:
- Spring Boot
- Spring Security (JWT)
- MySQL
- Swagger

## 📌 API Documentation
Swagger UI: http://localhost:8080/swagger-ui/index.html
